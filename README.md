# Final_Project: Aqua Forest
______________________________________________________________________________________________________________________
A simple 3D game where you find yourself in a forest completing quests for a ghost.
______________________________________________________________________________________________________________________
Controls:
- W/A/S/D for player movement
- Space to jump
- Shift to run
- E to collect items
______________________________________________________________________________________________________________________
Current gameplay objective is to collect items for the ghost (pick 3 plums).
